# Reflex-Icons
 This library provides all icons of React-icons wrapped for ReFlex framework

## Installation
```
pip install reflex-icons
```

## Upgradation
```
pip install reflex-icons --upgrade
```



#### [📖 View all Icons ](https://react-icons.github.io/react-icons)
